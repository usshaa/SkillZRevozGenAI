Self-Attention (Text→Text)
Every token attends to all previous tokens
LLM
Causal self-attention: Each query token computes Q·Kᵀ scores against all key tokens (in decoder: masked to past only). Softmax normalises → weighted sum of Values.

Causal mask: token at position i only sees positions 0…i

Cross-Modal Attention
Text tokens attend to image + table tokens too
MM
Full cross-modal attention: Text tokens can query all token types — text, image patches, table cells. No causal mask in encoder. Heads specialise by modality.

LLM Layer Stack
Uniform transformer blocks
LLM
Homogeneous stack: Every layer is identical in structure — MultiHead Self-Attention → Add&Norm → FFN → Add&Norm. Only the learned weights differ.

Multimodal Layer Stack
Specialised encoder + fusion layers
MM
Heterogeneous stack: Separate encoder towers per modality (text, vision, audio). Fusion layers merge representations. Late fusion via cross-attention bridges.

No Cross-Attention
Self-attention is the only mechanism
LLM
Pure self-attention: The LLM only has self-attention heads. All contextual understanding comes from tokens attending to other tokens in the same sequence. There is no separate K/V bank from another modality.
Q = K = V = same text sequence
🌉
Cross-Attention Bridge
Text queries image/table K,V banks
MM
Cross-attention: Text decoder queries (Q) attend to image encoder outputs (K, V) and/or table encoder outputs. This grounds textual generation in visual/structured evidence.
Q = text decoder · K,V = vision/table encoder
